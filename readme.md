# Spotify Music App - Assessment

### Requirements

- Docker and Docker-compose 3.9

### Features

- Nodejs + Express Backend
- MongoDB 4.2 database
- JWT Authentication
- Mongoose as an abstraction layer to the DB
- React frontend
- Spotify Frontend API Calls
